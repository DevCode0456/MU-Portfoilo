# 🚀 Ultra-Modern Animated Portfolio

**500% Animated | Dashboard UI | Framer Motion | Spline 3D | React**

A stunning, fully animated portfolio website built with React, Tailwind CSS, and Framer Motion — featuring dashboard-style UI, 3D elements, and smooth page transitions.

## ✨ Features

### 🎭 500% Animated Everything
- Framer Motion animations on every element
- Smooth page transitions with layout animations
- Interactive hover effects & micro-interactions
- Parallax scrolling effects
- Staggered entrance animations

### 🎨 Dashboard-Style UI
- No gradients — pure solid colors
- Modern, clean design language
- Card-based responsive layouts
- Perfect spacing, shadows & depth

### 🎯 Full Pages
- **Home** — Hero section with 3D Spline scene + animated sections
- **About Me** — Skills, experience & personal story
- **Projects** — Grid layout with detailed project pages
- **Contact** — Animated contact form & social links

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/DevCode0456/MU-Portfoilo.git

# Navigate to the project
cd MU-Portfoilo

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:3000`

## ⚙️ Customization

1. **Content** — Update `src/data/portfolioData.js` with your info
2. **3D Scenes** — Replace Spline scenes in components
3. **Colors** — Modify theme in `tailwind.config.js`
4. **Pages** — Add or edit pages in `src/pages/`

---

## 🌐 Deployment

```bash
# Build for production
npm run build

# Preview the build locally
npm run preview
```

Deploy the `dist/` folder to any static hosting: **Vercel**, **Netlify**, **GitHub Pages**, or **Cloudflare Pages**.
