# 🚀 Ultra-Modern Animated Portfolio

**500% Animated | Dashboard UI | Framer Motion | Spline 3D | React**

## ✨ Features

🎭 **500% Animated Everything**
- Framer Motion animations on every element
- Smooth page transitions
- Interactive hover effects
- Parallax scrolling
- Stagger animations

🎨 **Dashboard-Style UI**
- No gradients - pure solid colors
- Modern clean design
- Card-based layouts
- Perfect spacing & shadows

🎯 **Full Pages**
- Home (Hero + Sections)
- About Me
- Projects (with detail pages)
- Contact

🚀 **Technologies**
- React 18
- Tailwind CSS
- Framer Motion
- Spline 3D
- React Router
- React Icons

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📁 Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── data/          # Portfolio data
└── assets/        # Images, etc.
```

## 🎨 Theme Colors

**Dark** (Default):
- Background: #0a0a0f
- Surface: #12121a  
- Card: #1a1a24
- Primary: #0ea5e9

**Light**:
- Background: #f8fafc
- Surface: #ffffff
- Primary: #0ea5e9
- Accent: #a855f7

## 🛠️ Customization

1. Update `src/data/portfolioData.js`
2. Replace Spline scenes in components
3. Modify colors in `tailwind.config.js`

## 🌐 Deploy

```bash
npm run build
# Deploy dist/ folder
```
